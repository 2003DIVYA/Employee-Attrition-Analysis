# Employee-Attrition-Analysis
# Employee Attrition Analysis using Python, SQL & Power BI

## Project Overview

Employee attrition is one of the major challenges faced by organizations. High employee turnover increases recruitment cost, training expenses, and affects overall business productivity.

This project focuses on analyzing employee attrition data to identify the major reasons behind employee resignations and predicting attrition using Machine Learning.

The objective is to help HR teams understand employee behavior and take better retention decisions using data-driven insights.

---

## Problem Statement

To analyze employee data and identify the key factors responsible for employee attrition such as overtime, monthly income, job satisfaction, department, and work-life balance.

Also, to build a predictive model that can help organizations identify employees who are at high risk of leaving the company.

## Project Workflow

### 1. Data Collection

Downloaded the HR Analytics dataset from Kaggle.

### 2. Data Cleaning

Checked for:

* Missing values
* Duplicate records
* Incorrect data types

### 3. Exploratory Data Analysis (EDA)

Performed analysis on:

* Attrition Count
* Department-wise Attrition
* Overtime vs Attrition
* Job Satisfaction vs Attrition
* Monthly Income Distribution

### 4. Data Preprocessing

Used Label Encoding to convert categorical columns into numerical values for Machine Learning.

### 5. Machine Learning Model

Built Employee Attrition Prediction model using:

* Random Forest Classifier

### 6. Model Evaluation

Evaluated the model using:

* Accuracy Score
* Classification Report
* Confusion Matrix

### 7. Feature Importance Analysis

Identified the top factors responsible for employee attrition.

### 8. SQL Queries

Created SQL queries for:

* Total Employees
* Attrition Count
* Department-wise Analysis
* Average Salary by Department
* Overtime Analysis

### 9. Power BI Dashboard

Designed an interactive HR Dashboard with:

* Total Employees
* Attrition Rate
* Department-wise Attrition
* Overtime Analysis
* Job Satisfaction Analysis

---

## Model Performance

### Accuracy Achieved

**86.73%**

The Random Forest model performed well in predicting employee attrition and identifying high-risk employee groups.

---

## Key Business Insights

* Employees working overtime showed higher attrition
* Lower monthly income increased resignation chances
* Low job satisfaction strongly impacted employee retention
* Sales department showed higher attrition compared to other departments

These insights can help HR teams improve employee retention strategies.
